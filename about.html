So What's Up With This Website?

        
If you're wondering what this site is about, you've come to the right place! Isaiah's Music is a website created by Isaiah Henry as a project to showcase front end web development fundamentals.

        
So Where Does The Music Come From?

        
 All of the music on this site was made by Isaiah Henry using a digital audio workstation (DAW) call FL Studio. All songs that are on this website were made solely to be presented here.

        
How Was This Website Made?

        
This website was made using HTML5  CSS3  and Javascript 
        

        
The main functionality of this site comes from the MUSIC page. That page uses Javascript to populate a table of songs based on a 2 dimnesional array containing the necessary properties that correspond to the songs that will be played. When you click on a row of the table, there is a Javascript function that replaces the music player with one with the corresponding attributes of the selected song. There is also a form that allows you to filter through the genres. There is a function connected to this form that prevents it from reloading the page and just replace the data in the table to display only the songs with the corresponding genre. There is also Javascript code that uses an api called Audio Context that makes it possible to make a custom "audio tag" with personalized controls.
